    Managed Apache Kafka in the Cloud

    Oracle Event Hub Cloud Service delivers the power of Kafka as a managed streaming data platform integrated with the rest of Oracle’s Cloud.
     

    Fast

    Instant creation, scaling of Topics – ready for action in seconds
     

    Managed

    Oracle manages the Kafka infrastructure, while you leverage the power and simplicity of the platform
     

    Flexible

    Integrate either using REST APIs or Native-Kafka APIs
     

    Cost Effective

    Scale from a few hundred operations a second to a few million operations per second
